# рЃЏрЃгрЃЋрЃљрЃЋрЃћ рЃЎрЃЮрЃюрЃцрЃџрЃўрЃЦрЃбрЃў рЃЮрЃЌрЃљрЃ«рЃерЃў

рЃЉрЃљрЃюрЃљрЃЎрЃўрЃА рЃЏрЃћрЃЮрЃЌрЃ«рЃћ рЃЊрЃдрЃћрЃљ рЃЊрЃљ рЃљрЃдрЃЏрЃЮрЃЋрЃљрЃЕрЃўрЃюрЃћрЃЌ, рЃарЃЮрЃЏ рЃЮрЃЌрЃљрЃ«рЃерЃў рЃћрЃарЃЌ-рЃћрЃарЃЌ рЃЉрЃљрЃЋрЃерЃЋрЃА рЃцрЃўрЃќрЃўрЃЎрЃБрЃарЃљрЃЊ рЃњрЃљрЃБрЃАрЃгрЃЮрЃарЃЊрЃюрЃћрЃю, 3 рЃЉрЃўрЃГрЃў рЃЕрЃўрЃЋрЃўрЃА рЃарЃЮрЃЏ рЃЏрЃћрЃЮрЃЌрЃ«рЃћ рЃАрЃфрЃћрЃЏрЃћрЃА рЃўрЃЏрЃўрЃбрЃЮрЃЏ, рЃарЃЮрЃЏ рЃўрЃА рЃЏрЃљрЃЌ рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃљрЃЊ рЃљрЃДрЃћрЃюрЃћрЃЉрЃЊрЃљ рЃАрЃўрЃбрЃДрЃЋрЃўрЃћрЃа рЃерЃћрЃБрЃарЃљрЃфрЃ«рЃ░рЃДрЃЮрЃцрЃљрЃА, рЃљрЃерЃўрЃюрЃћрЃЉрЃЊрЃљ, рЃљрЃДрЃБрЃарЃћрЃЉрЃўрЃюрЃћрЃЉрЃЊрЃљ рЃљрЃАрЃљрЃЎрЃўрЃАрЃЌрЃЋрЃўрЃА рЃерЃћрЃБрЃцрЃћрЃарЃћрЃЉрЃћрЃџ рЃЋрЃўрЃЊрЃћрЃЮрЃћрЃЉрЃА рЃЊрЃљ рЃњрЃљрЃЋрЃЏрЃгрЃљрЃарЃЊрЃўрЃЌ рЃЊрЃљ рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃљрЃАрЃћрЃЌрЃў рЃерЃћрЃБрЃарЃљрЃфрЃ«рЃ░рЃДрЃЮрЃцрЃўрЃА рЃЏрЃћрЃарЃћ рЃЌрЃљрЃЋрЃў рЃЋрЃћрЃдрЃљрЃа рЃерЃћрЃЋрЃўрЃЎрЃљрЃЋрЃћрЃЌ, рЃЊрЃљ рЃЊрЃљрЃЋрЃљрЃарЃбрЃДрЃўрЃЌрЃЮ; рЃњрЃБрЃюрЃЊрЃерЃў рЃўрЃАрЃўрЃюрЃў рЃљрЃа рЃўрЃЦрЃфрЃћрЃЮрЃЊрЃюрЃћрЃю рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃўрЃА рЃЏрЃўрЃЏрЃљрЃарЃЌ рЃЏрЃбрЃарЃБрЃџрЃљрЃЊ рЃЊрЃљ рЃљрЃЏ рЃЎрЃЮрЃюрЃцрЃџрЃўрЃЦрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃњрЃљрЃЋрЃўрЃњрЃћрЃЌ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃфрЃћрЃЏрЃўрЃА рЃцрЃљрЃЦрЃбрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ.
